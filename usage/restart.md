# Restart

### When to Restart <a href="#when-to-restart" id="when-to-restart"></a>

* Tokens updated
* Chat IDs changed
* Keywords updated
* Features toggled

***

### [​](https://meeseeks.mintlify.app/restart#restart-methods)Restart Methods <a href="#restart-methods" id="restart-methods"></a>

#### [​](https://meeseeks.mintlify.app/restart#command)Command <a href="#command" id="command"></a>

```
/restartbot
```

#### Menu

Use the Restart Bot button.

{% hint style="info" %}
Changes will not apply until the bot is restarted.
{% endhint %}
