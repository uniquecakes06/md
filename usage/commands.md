# Commands

### Available Commands <a href="#available-commands" id="available-commands"></a>

| Command       | Description         |
| ------------- | ------------------- |
| `/start`      | Open welcome screen |
| `/menu`       | Open settings menu  |
| `/status`     | Check bot status    |
| `/startbot`   | Start the bot       |
| `/stopbot`    | Stop the bot        |
| `/restartbot` | Restart the bot     |

{% hint style="info" icon="lightbulb-exclamation" %}
Use `/status` to quickly debug issues.
{% endhint %}
