# Troubleshooting

### &#x20;No Discord user token configured <a href="#no-discord-user-token-configured" id="no-discord-user-token-configured"></a>

{% hint style="warning" %}
Add at least one token in Discord Tokens.
{% endhint %}

***

### [​](https://meeseeks.mintlify.app/troubleshooting#configure-at-least-one-telegram-bot-token)Configure at least one Telegram bot token <a href="#configure-at-least-one-telegram-bot-token" id="configure-at-least-one-telegram-bot-token"></a>

{% hint style="warning" %}
Set Token 1 in Telegram Tokens.
{% endhint %}

***

### [​](https://meeseeks.mintlify.app/troubleshooting#at-least-one-feature-must-be-on)At least one feature must be on <a href="#at-least-one-feature-must-be-on" id="at-least-one-feature-must-be-on"></a>

Enable one of:

* Messages
* Member Joins
* Tickets

***

### [​](https://meeseeks.mintlify.app/troubleshooting#no-forwarded-messages)No forwarded messages <a href="#no-forwarded-messages" id="no-forwarded-messages"></a>

Check:

* Message feature is ON
* Keywords are set
* Message Chat ID is correct
* Bot was restarted

{% hint style="info" %}
Most issues are fixed by restarting the bot.
{% endhint %}
