# Introduction

### What You Can Do

* Forward messages based on keywords
* Track member joins
* Monitor tickets

{% hint style="info" %}
All configuration is done through Telegram — no dashboard required.
{% endhint %}

### Quickstart

Follow these steps:

1. Add Discord token(s)
2. Add Telegram bot token(s)
3. Set chat IDs
4. Add keywords
5. Enable features
6. Restart the bot
